# SmartLibrary

A console-based application designed to manage library operations efficiently.

# Features
* View available books
* Borrow and return books
* Manage user profiles
* Catalog & book management
* Technical Higlights

#  How to Run the Application
1. Download or clone this repository.
2. Open the project folder in your preferred IDE (like IntelliJ IDEA, since the project includes `.idea` files).
3. Locate the main file inside the `src` folder.
4. Run the application and follow the on-screen console prompts.
